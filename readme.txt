Crea formularios personalizados para generar y descargar certificados en PDF a partir de datos en Google Sheets.

== Description ==

El plugin "Generador de Certificados PDF" te permite crear formularios personalizados para la búsqueda y descarga de certificados en PDF generados a partir de datos almacenados en Google Sheets.

**Características principales:**

* Interfaz de configuración visual para el diseño de certificados
* Conexión con Google Sheets para obtener datos
* Generación de formularios personalizados mediante shortcodes
* Validación y búsqueda de registros
* Generación automática de PDFs personalizados
* Soporte para múltiples certificados y plantillas

== Installation ==

1. Sube la carpeta `generador-certificados-pdf` al directorio `/wp-content/plugins/`, o instala el plugin a través del instalador de plugins de WordPress.
2. Activa el plugin desde el menú 'Plugins' en WordPress.
3. Configura tu API Key de Google en la página "Certificados PDF" > "Configuración".
4. Crea tu primer certificado en "Certificados PDF" > "Añadir Nuevo".
5. Utiliza el shortcode proporcionado para mostrar el formulario en cualquier página o entrada.

== Frequently Asked Questions ==

= ¿Necesito una cuenta de Google Cloud para usar este plugin? =

Sí, necesitas una cuenta de Google Cloud para obtener una API Key que permita la conexión con Google Sheets. El plugin incluye instrucciones detalladas sobre cómo obtener esta API Key.

= ¿Puedo tener múltiples formularios en la misma página? =

Sí, puedes insertar múltiples shortcodes en la misma página para mostrar diferentes formularios de certificados.

= ¿Qué formatos de plantilla puedo usar? =

Puedes usar imágenes en formato PNG, JPG, o PDF como plantillas para tus certificados.
